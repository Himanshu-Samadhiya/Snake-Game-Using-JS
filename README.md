# Snake Game 🐍

A simple and interactive Snake Game built using HTML, CSS, and JavaScript. The game features smooth snake movement, score tracking, high score saving using local storage, food generation, collision detection, timer functionality, and restart support.

## Features

* 🎮 Smooth snake movement using keyboard controls
* 🍎 Random food generation
* 📈 Score and High Score tracking
* 💾 High score saved using Local Storage
* ⏱️ Live timer functionality
* 💥 Wall and self-collision detection
* 🔄 Restart game option
* 🎨 Responsive and clean UI design

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Controls

* ⬆️ Arrow Up → Move Up
* ⬇️ Arrow Down → Move Down
* ⬅️ Arrow Left → Move Left
* ➡️ Arrow Right → Move Right

## Project Purpose

This project was created to practice JavaScript DOM manipulation, game logic, event handling, CSS Grid, and local storage implementation.
